[Blog](https://thejiahao.github.io) site built with Astro + React + UnoCSS.

## Main features

- Multilanguage
- Responsive mobile and desktop layout
- RSS
- JSON-LD schema
- Blog search
- Dark/light themes
- Table of contents with highlight
- Comments
- Search engine indexing workflow
- Typst equations support
