Windows 下通过 Scoop 为 VS Code 配置 clangd

2022-05-12
阅读时长: 1 分钟

使用 Scoop 安装 clangd 可以自动配置环境变量和避免 Visual Studio Code 找不到 #include。 本文介绍了通过 Scoop 安装 clangd 的方法。

# 安装方式

安装 Scoop

Invoke-WebRequest get.scoop.sh | Invoke-Expression

安装 Winlibs 编译的 MinGW + Clang + clangd

scoop install mingw-winlibs

VS Code 安装 clangd 扩展。
